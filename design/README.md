# طراحی گرافیکی مطلک

## وظایف طراح :

- [x] آشنایی اولیه با پروژه
- [ ] ساخت صفحه ثبت نام ( ۲۱ خرداد -> ۲۵ خرداد)
- [ ] ساخت صفحه ورود (۲۶ خرداد -> ۲۹ خرداد)
